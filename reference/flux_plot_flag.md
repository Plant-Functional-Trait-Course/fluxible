# creates the flag column to be used by flux_plot

creates a column with quality flags (from flux_quality) for the part of
the rows to be kept, and cut flag for rows to be discarded

## Usage

``` r
flux_plot_flag(slopes_df, param_df)
```

## Arguments

- slopes_df:

  as provided in flux_plot

- param_df:

  as provided by flux_param
