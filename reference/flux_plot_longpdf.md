# print plots in a single long pdf

print plots in a single long pdf

## Usage

``` r
flux_plot_longpdf(f_plot, f_plotname, nb_fluxid, longpdf_args)
```

## Arguments

- f_plot:

  plot from flux_plots

- f_plotname:

  name of plot

- nb_fluxid:

  number of facets

- longpdf_args:

  arguments for longpdf in the form `list(ncol, width (in cm), ratio)`
