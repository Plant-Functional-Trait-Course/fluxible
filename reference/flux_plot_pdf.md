# print plots in pdf

print plots in pdf

## Usage

``` r
flux_plot_pdf(f_plot, f_plotname, facet_wrap_args, nb_fluxid)
```

## Arguments

- f_plot:

  plot from flux_plot

- f_plotname:

  name of plot

- facet_wrap_args:

  list of arguments for
  [facet_wrap_paginate](https://ggforce.data-imaginist.com/reference/facet_wrap_paginate.html)

- nb_fluxid:

  number of facets
